Материал урока:

Выравнивание Картинки в нутри элемента нужно осознавать понятия:

"span" - Строчный элемент

"div" - Блочный элемент

img - строчный элемент, - зангимает место столько сколько размер картинки.

    Для выравнивания картинки по центру прописываем "картинке - img" свойства в CSS:
  • display: block;
  • margin: 0 auto;
        img{
          display: block;
          margin: 0 auto;
        }

Так же можно выравнивать картинку в блоке:

Создаем блок: "container" в него помещаем картинку: - "Фото Бота.jpg"

Выравниваем сам блок на середину документа:

  • Устанавливаем блоку ширину width:
  • Устанавливаем блоку margin:
  • Устанавливаем блоку background-color:
        .container {
          width: 800px;
          margin: 0 auto;
          background-color: #f48033;
        }

Выравниваем картинку в блоке 3 способа:

  1. Как выше описывалось,задаем свойство картинке: "display: block;" "margin: 0 auto;" 
              img {
            display: block;
            margin: 0 auto;
          }
  2. Картинка остается строчной, задаем свойство контейнеру: text-align: center; 
              .container {
            background-color: #f48033;
            width: 800px;
            margin: 0 auto;
            text-align: center;
          }
  3. Сделать контейнер флексовым "display: flex": 
                .container {
              display: flex;
              justify-content: center;
              width: 600px;
              margin: 0 auto;
              background-color: #f48033;
            }
  4. Если установить в одном flex контейнере 2 картинки они станут вряд а не в столбец. поэтому нужно применить св-во: "flex-direction: column;", а что бы картинки не растягивались вдоль, нужно установить для "img" св-во: "align-self: center"

Если хотим прижать картинку внутри блока к правому краю, то нужно прописать картинке стили:

    img {
  • display: block;
  • margin: 0 0 0 auto;
    • }

Что бы прижать картинку с лева внутри блока не нужно указывать ни каких отступов для картинки, эта позиция идет по умолчанию >

10-ть способов как можно выровнять один элемент в нутри другого по Центру (по Горизонтали и по Вертикали).

Есть 2 Дива, Родительский - "parent p1" и Дочерний - "child c1", один в нутри другого.

Рассмотрим написание стилей для выравнивания по центру:

    Для начала установим "p1" - "position: relative;" "c1" - "position: absolute;"
    1-вый вариант: "position" transform: translate()
  1. Для дочернего элемента укажем свойства: 
                .c1 {
              position: absolute;
              left: 50%;
              top: 50%;
              transform: translate(-50%,-50%);
            }
    • transform: translate(-50%,-50%); Двигает элемент относительно своего центра.
    • left: 50%; - Двигает элемент относительно своего левого верхнего угла с лева на право
    • top: 50%; - Двигает элемент относительно своего левого верхнего угла с верху вниз
    2. 2-й вариант: "position" top/ left/ right/ bottom
  2. Так же для дочернего элемента укажем стили. 
                  .c1 {
              position: absolute;
              left: 0;
              top: 0;
              right: 0;
              bottom: 0;
              margin: auto;
            }
    3. 3-й вариант: "position" inset: / margin
  3. Устанавливаем вместо 4х свойств left: 0; top: 0; right: 0; bottom: 0; Одно свойство: inset: 0; 
                  .c1 {
                position: absolute;
                inset: 0;
                margin: auto;
              }
    4. 4-й вариант: "flex-box" justify-content
  4. Теперь устанавливаем свойства display: flex; для Родительского элемента: 
                  .p1 {
                display: flex;
                justify-content: center;
                align-items: center;
              }
      4.a) Выравнивание так же происходит засчет родительского display:flax; и дочернего align-self: center; 
    •                   .p1 {
                    display: flex;
                    justify-content: center;
                  }
                  .c1{

  5. Следующий способ тоже через display:flax, но задаются другие стили:

  6. С изпользованием display: grid;
